# Datasets  
  
The datasets used in this project are not distributed with the repository.  
Please download them manually from their official sources.  
  
## WebNLG  
  
Knowledge graph-to-text dataset used for structured data generation tasks.  
  
Repository:  
https://gitlab.com/shimorina/webnlg-dataset  
  
## GenWiki  
  
Dataset for generating Wikipedia-style text from knowledge graphs.  
  
Repository:  
https://github.com/zhijing-jin/genwiki  
  
## TekGen  
  
Large-scale text-triple generation dataset derived from Wikipedia and Wikidata.  
  
Repository:  
https://github.com/google-research-datasets/KELM-corpus
