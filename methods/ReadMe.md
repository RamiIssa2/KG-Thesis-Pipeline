# Evaluated Methods

This project evaluates several graph-to-text generation methods.
The original papers and official implementations are listed below.

## CycleGT

Graph-aware cycle training framework for graph-to-text generation.

Paper:
https://arxiv.org/pdf/2006.04702

Repository:
https://github.com/QipengGuo/CycleGT

## Grapher

Instruction-tuned graph reasoning and generation model.

Paper:
https://arxiv.org/pdf/2211.10511

Repository:
https://github.com/IBM/Grapher

## KGGen

Knowledge graph-conditioned text generation framework.

Paper:
https://arxiv.org/pdf/2502.09956

Repository:
https://github.com/stair-lab/kg-gen
